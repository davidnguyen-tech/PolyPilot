Quick Start

Up and running in minutes

01

Install Prerequisites

Get .NET 10 SDK and the MAUI workload installed.

dotnet workload install maui
02

Clone & Build

Clone the repo and build for your platform.

git clone https://github.com/PureWeen/PolyPilot.git
cd PolyPilot
dotnet build -f net10.0-maccatalyst
03

Launch & Command

Open the app and start orchestrating your Copilot agents.

open bin/Debug/net10.0-maccatalyst/
  maccatalyst-arm64/PolyPilot.app
